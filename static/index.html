AI補正付き文字起こし

音声や動画ファイルからテキストへ変換し、読みやすい文章に自動補正します。

YouTubeからの取り込み

YouTubeのURLを指定すると、動画から音声を取得して文字起こしを行います。

または

音声ファイルをアップロード

または、このエリアにファイルをドラッグ＆ドロップしてください。
対応形式: MP3 / WAV / M4A / OGG / FLAC

カンマ区切りで入力すると、専門用語や固有名詞の認識精度が向上します。

精度と処理速度のトレードオフがあります。通常は「Base」で十分です。
